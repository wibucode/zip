# zip
Zip BruteForce 

# Installation
$ pkg install python2
$ git clone https://github.com/wibucode/zip

# Cara menggunakan
$ ketik help untuk menampilkan bantuan (command)

$ ketik start untuk memulai bruteforce

  Contoh: start game.zip

$ ketik wlch untuk ganti wordlist

  Contoh: wlch wordlist.txt
